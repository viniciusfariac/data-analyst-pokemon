# API Data Collector (Pokémon) 
Simple project for studying Python and data analysis. 
### Features 
- Consumes the PokéAPI API 
- Collects Pokémon data (name, id, height, weight, experience) 
- Clean structure in DataFrame (Pandas) 
- Data cleaning functions 
- Export to CSV 
- Ready to run locally 
### How to run
```bash
pip install -r requirements.txt
python main.py
